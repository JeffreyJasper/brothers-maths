# 扑傻瓜遊戲

## 專案概覽

這是一個使用 React 建立的「扑傻瓜」網頁遊戲。

## 核心功能

*   **遊戲目標:** 在限定時間內獲得 10 分。
*   **計分規則:**
    *   打中沒有頭髮的男人：+1 分。
    *   打中有頭髮的男人：-1 分。
*   **時間限制:** 60 秒。
*   **遊戲結束:**
    *   時間到。
    *   分數達到 10 分。

## 遊戲元素

*   **角色:**
    *   分為兩種：有頭髮的男人 (👨) 和沒有頭髮的男人 (👨‍🦲)。
    *   會隨機在 9 個洞中出現。
*   **遊戲介面:**
    *   顯示目前分數。
    *   顯示剩餘時間。
    *   遊戲開始按鈕。
    *   遊戲結束時顯示結果。

## 技術棧

*   React
*   Vite
*   CSS
